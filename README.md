# Esp32 Dongle with Custom PCB Antenna
A unique ESP32 C3 based PCB design with on board PCB antenna tuned for 2.4GHz Wifi and BLE.

## PCB Design Tool
---> Altium Designer

## Design Details

- MCU: ESP32-C3  with BMI55
- Wireless: 2.4 GHz WiFi + BLE  
- Antenna: Custom on-board PCB antenna (impedance matched and tuned)  
- PCB Design Tool: Altium Designer
